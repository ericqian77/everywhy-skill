# Quality rubric — questions for the final pass

Walk this after the course renders correctly (SKILL.md step 7) and before writing the delivery note (step 8). These are judgment questions, not a scoring sheet — if one lands badly, that's a revision, not a footnote. Adapted from the V2 design review; kept as questions on purpose.

## Understanding
- Can the target learner state the core idea in their own words after one pass, without narration beyond the page?
- Does the final reassembly *show* the answer, or does a caption merely assert it?
- Is every stage necessary? Which one would you cut if forced — and why haven't you?

## Correctness
- Could a skeptical reviewer re-derive every number and geometric relation from the comments you left?
- Does any analogy quietly teach something false at the edges? (Say where it breaks, or soften it.)

## Audience fit
- Is every new symbol or term *needed* before it appears — did the learner feel the gap it fills?
- Does the language complexity serve the concept, or is it just how adults write?
- Are the stated prerequisites the real ones?

## Visual reasoning
- Does each moving element carry explanatory weight, or is some of it decoration?
- Does rendered geometry literally match the data (lengths, areas, angles to scale)?
- Can the learner stop at any key state and inspect it, or does the page rush past?

## Interaction
- Does interacting *change understanding*, or just change pixels?
- Are the autoplay path and the manual path both coherent from start to end?

## Originality
- Was this visual language chosen *for this concept*, or inherited from an example?
- If someone lined this up beside the existing courses, would it read as its own thing?
